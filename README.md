Краткое описание:
Простое веб-приложение «Hello World» с двумя кнопками, которые меняют цвет текста на зелёный и оранжевый. 
Проект сделан как статическая страница и опубликован через GitHub Pages.

Функциональность
- Отображает заголовок Hello World.
- Кнопка «Зелёный» меняет цвет текста на зелёный.
- Кнопка «Оранжевый» меняет цвет текста на оранжевый.
- Есть плавная анимация смены цвета.

Технологии
- HTML5 — разметка страницы.
- CSS3 — стилизация и анимация.
- JavaScript (Vanilla) — обработчики кликов по кнопкам.
Фреймворки не используются, потому что логика минимальная и полностью выполняется в браузере.

Почему нет бэкенда
Бэкенд не нужен, потому что приложение:
- не хранит данные
- не использует базу данных

Структура проекта
- `index.html` — вся логика и стили приложения в одном файле.
